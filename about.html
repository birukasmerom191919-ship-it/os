About This Website

This project is designed to help students understand what happens from the moment a computer is powered on until the operating system becomes ready.

The website presents the boot process in step-by-step form, with visual sections and practical explanations. It covers key concepts like power-on initialization, firmware behavior, bootloader selection, and kernel startup.

My goal was to make technical ideas easier to learn by combining short theory, comparison between BIOS and UEFI, and examples of common boot failures.
